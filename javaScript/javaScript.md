## 1. What is JavaScript?

JavaScript (JS) is a **high-level**, **interpreted**, **dynamic programming language**.  
It is primarily used for:

- Web development (frontend + backend via Node.js)  
- Interactive websites and UI  
- Server-side scripting (Node.js)  
- Game development  
- Mobile app development (React Native, Ionic)  

JavaScript is famous because it **runs in browsers**, is **event-driven**, and **very versatile**.

Example:

console.log("Hello, world!"); // prints message to console

## 3. How JavaScript Runs

JavaScript is an **interpreted language**, but modern engines use **Just-In-Time (JIT) compilation**.

### How JS Executes Code (Step by Step)

1. **You write `.js` code**  
2. The browser or Node.js reads your code  
3. JS engine converts code into **machine code** at runtime (JIT compilation)  
4. Code executes immediately inside the **engine environment** (V8 in Chrome, SpiderMonkey in Firefox)

This makes JS **fast, dynamic, and flexible** while running in any browser or runtime.

---------------------------------------------------------------------

## JavaScript Engines

### 1. V8 (Google Chrome, Node.js)

- Written in C++  
- Uses JIT compilation â†’ converts JS to machine code on the fly  
- Optimized for performance in modern apps

### 2. SpiderMonkey (Firefox)

- First JS engine created by Brendan Eich  
- Implements ECMAScript standard  
- Includes JIT compilation for speed

### 3. JavaScriptCore (Safari)

- Also called **Nitro**  
- Apple's JS engine for Safari and WebKit

## Why JavaScript Works This Way

JS focuses on:

- **immediacy** â†’ code runs directly in browsers  
- **dynamic behavior** â†’ no need to declare types  
- **asynchronous operations** â†’ callbacks, promises, async/await

If JS were fully compiled like C:

- Web interactivity would be slower to update  
- Dynamic features like DOM manipulation and event handling would be harder  
- Beginners would face a steeper learning curve

JS balances **speed**, **flexibility**, and **developer-friendliness**, making it perfect for web development beginners.
